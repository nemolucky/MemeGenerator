Контрольная работа предполагает закрепить понимание принципов работы Express.js, умение создавать сервер, маршруты, работать с middleware, обрабатывать запросы и структурировать backend-приложение.
Работа должна продемонстрировать способность студента применять Express.js для обслуживания простого API, взаимодействия с данными и организации архитектуры проекта.
Студент должен разработать полноценное Express-приложение, которое взаимодействует с данными и предоставляет API для одного из вариантов ниже (выбрать только один пункт из списка 1–37).
Приложение должно включать:
Базовый Express-сервер.
Маршруты (GET, POST; PUT/DELETE — при необходимости).
Работа с параметрами (req.params), query-параметрами (req.query).
Обработка тела запроса через express.json() и/или express.urlencoded().
Минимум один собственный middleware.
Раздачу статических файлов через express.static().
Логику, разнесённую по модулям (routes + controllers).
Структурированный репозиторий GitHub с README и скриншотами.
